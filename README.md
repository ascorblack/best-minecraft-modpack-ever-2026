## Установка

### Клиент
Скачай `*-client.mrpack` из [Releases](../../releases) и импортируй в **Modrinth App** или **Prism Launcher** (*Add Instance → Import*). Лаунчер сам поставит NeoForge и все моды.
Либо `*-curseforge.zip` — для лаунчера CurseForge/Overwolf, ATLauncher, Prism.

### Сервер
Скачай `*-server.zip` из [Releases](../../releases). Внутри — папка `mods/` (серверный набор) и инструкция. Коротко:
1. Установи сервер NeoForge 21.1.242 (`neoforge-21.1.242-installer.jar --install-server .`).
2. Распакуй `mods/` из архива в папку сервера.
3. `eula=true`, запусти `./run.sh nogui`.

> ⚠️ **Важно:** на выделенный сервер нельзя класть клиентские рендер-моды (Sodium 0.8.x, Iris и др.) — новый Sodium падает на старте сервера (`org.lwjgl.Version`). Серверный набор в `*-server.zip` уже очищен от них. Если собираешь сервер из `.mrpack` — его серверный профиль исключает клиентские моды автоматически.
